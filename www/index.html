The JEE Docker Application stack is a local environment to develop Java Enterprise Applications.

Services

The following services are installed:

  • nginx www server on port 80
  • wildfly application server on port 8080
  • wildfly management console on port 9990
  • wildfly remote debugging on port 8787
  • phpMyAdmin on port 5050
  • mysql database on port 3306

Passwords

The username for all services is root and the password for all services is also root.

Source Code

You can find more information and download the source code here.

JPA Datasource mydb

During build a database mydb is created in the mysql server. This database is added as a JNDI data source to the application server. The JNDI name is java:jboss/datasources/mydb
For details see datasources in the wildfly admin console.

Note: You can debug the application server as a Remote Java Application on port 8080
Note: You can deploy your war directly to wildfly using the eclipse jboss plugin.
Note: Copy your HTML/javascript/css to the www subfolder (the folder where this file currenlty resides). Then you can do rest calls directly to http://localhost/srv to reach your wildfly application server in the same domain (no CORS headers required).
